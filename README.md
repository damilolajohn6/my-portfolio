# Portfolio

This is a private portfolio project.

## Description

This project is a portfolio showcasing my skills and projects as a software engineer.

## Installation

To install the dependencies, run:

```bash
npm install
```

## Usage

To start the development server, run:

```bash
npm run dev
```

To build the project, run:

```bash
npm run build
```

To lint the project, run:

```bash
npm run lint
```

To preview the built project, run:

```bash
npm run preview
```
